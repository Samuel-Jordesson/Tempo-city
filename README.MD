## O projeto

o projeto foi criando para fins de estudos

o proejto ele é pra fins didadicos então resolvi mandar pro git com todas as pastas e arquivos, incluindo ate a pasta .env
assim quem for baixar esse sistema pra tetar ja ta tudo configrado pra rodar localmente.

simples de testar
basta baixar o arquivos 

entre na pasta e execulte npm install
se por acaso vc não tiver o node instalado no seu pc, baixe no site https://nodejs.org/en/download

logo depois rode no terminal, node server.js para iniciar o servidor no http://localhost:8080

depois abra o arquvo index.html

assim vc pode pesquisar pelo nome da sua cidade e verificar o tempo em tempo real
